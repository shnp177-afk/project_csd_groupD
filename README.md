#  Codify Learning Platform

A responsive educational website prototype for Codify, built using HTML, CSS, JavaScript, and Bootstrap. The project includes a polished homepage, a membership page, and interactive cookie consent behavior.

## Features

- Modern hero section with background imagery and a floating highlight card
- Section-based layout for courses, bootcamps, quizzes, and tutor introductions
- Feature cards with custom images for pacing, certification, and instructors
- Membership page with plan selection and signup form
- Cookie consent popup with accept/reject flow and warning dialog
- Fully responsive design for desktop and mobile screens

## Project Structure

- `index.html` — Homepage with hero, course/bootcamp/quiz/tutor sections, and feature cards
- `membership.html` — Membership plan page with signup form and plan cards
- `css/mainstyle.css` — Global styles for navigation, layout, hero, cards, and popup UI
- `css/membership.css` — Membership page-specific styles and hero layout
- `js/index.js` — Cookie consent popup logic for the homepage
- `js/membership.js` — Membership plan rendering and form validation logic
- `images/` — Visual assets used throughout the site
- `faviconio-logo/` — Logo and favicon files

## How to Use

1. Open `index.html` in a browser to view the homepage.
2. Navigate to `membership.html` to see the membership plan page.
3. Review the `css/` and `js/` files to inspect styling and interactive behavior.

## Notes

- The cookie popup is intentionally shown on every page load.
- The membership hero section now uses a side-by-side text and image layout.
- Images are displayed without cropping to preserve original quality.
